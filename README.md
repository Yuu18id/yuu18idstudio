# Yuu18id Studio

A portfolio and commission landing page for custom Minecraft skin design and pixel art services.

## Features

- 3D skin viewer powered by skinview3d
- Portfolio gallery with category filter
- Scroll animations via AOS

## Built With

- HTML, CSS, JavaScript
- Bootstrap 5
- [skinview3d](https://github.com/bs-community/skinview3d)
- [AOS](https://michalsnik.github.io/aos/)
